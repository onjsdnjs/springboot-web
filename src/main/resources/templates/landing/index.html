Coinext는 무엇이 다른가?

기존 거래소들의 보안/안전에 대한 문제점들을 보완하였습니다.
개발자가 직접 만들고 운영하는 사이트입니다. 빠른 피드백으로 견고한 시스템을 만들겠습니다.

강력한 보안기술

Coinext는 유저의정보를 모두암호화하여, 블럭체인상에 등록하여 사용합니다.
사용자의 유니크한 디바이스의 정보를 암호화에 사용하여, 사용자가 허락한 기계에서만 로그인이 가능하게합니다. 외부의 접근할수있는 모든 데이터들은 모두 암호화하여, 데이터베이스에 저장합니다.
사용자를 특정지을수없는 모호한 보안매체는 사용하지않습니다.예) SMS
사용기술 : ipfs, client-js, dapp

확장/성능

Coinext는 모든기능이 클라우드 플랫폼에서 멀티프로세스로 동작하게됩니다.
언제든 성능이 필요할땐, 확장이 가능한 거래소입니다.
사용자들은 언제나 쾌적한 거래환경을 경험하실 수 있습니다.
사용기술: aws, messagequeue, websocket, redis

거래소간의 연동

거래소간의 연동을 통해, 거래가/시장가의 데이터를 연동하는 허브역할을 함으로써 가상화폐의 안정적인 가격에 기여할수 있습니다.
다양한 API를 제공하여, 보다 편리하고 안전한 거래를 할수있습니다.

Coinext 회사소개

가상화폐를 거래하던 개발자가 기존거래소문제들로 너무 답답해서 만든 거래소개 1인 개발회사입니다.
평상시 나라면 이렇게 만들겠어라는 생각들을 직접 구현한 시스템입니다.

  • 항상, 보안과 확장성에 일순위로 생각하며 개발하였습니다.
  • 시스템의 구조와 문제점을 오픈하고, 오픈소스로 진행하여 좀더 나은 거래소를 만들예정입니다.

FAQ

사용자가 자주하는 궁금한 질문에 답을 모았습니다.

정말 혼자 1인개발하신 건가요?

네, 다년간의 경험과 지식을 토대로 만든 솔류션입니다.

성능면에서 강점이 있나요?

내부적 구현이 rx형식의 async구조와 큐, 그리고 cloud에서 scale out하게 확장할수있는 구조로 구현이 되어있습니다.

제작시 사용한 언어와 기타 라이브러리등은 뭐가있나요 ?

java1.8로 구현되어있고, 래빗엠큐와 레디스를 기본적으로 사용하고 있고, was는 스프링부트를 사용합니다.

향후 추가할 서비스가 있나요?

네, 마진거래나 엣지, 보험상품 서비스를 추가하고 싶습니다.
